# csc125-final-project-or-similar-
My Python final project for CSC-125"
## Project Description
This Python program is a simple restaurant ordering system. It shows a menu, lets the customer enter items and quantities, checks for invalid inputs and stores the order in a dictionary. The program then calculates the subtotal, tax and total and prints a clear receipt with all order details.
## Technical Requirements
- Dictionary with at least 3 key-value pairs
- Custom function with parameters/return value
- Loop (for or while)
- Conditional logic (if/elif/else)
- User input with validation
- Clear, meaningful output
